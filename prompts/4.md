# Prompt 4: Build, Deploy and Test Quadratic Dutch Auction

## Goal
Build the updated Solver SDK and test the quadratic curve implementation end-to-end.

## Step 1: Build Solver SDK
```bash
cd solver-sdk && npm run build
```

Verify no TypeScript errors.

## Step 2: Run Move Tests
```bash
cd contracts && movement move test --filter dutch
```

Ensure all Dutch auction tests pass with quadratic formula.
